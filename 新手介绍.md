# 欢迎来到《斑木湾》

你们是命案专家，一群在推理小说俱乐部聚了十年的老太太。现在你们撞上了真正的谋杀案，背后还藏着比凶手更黑暗的东西。

# 掷骰

掷2D6加能力调整值，看结果：

- 6以下：失败，严重后果，守秘人行动
- 7-9：做到了，但要付出代价
- 10-11：困难成功，干净利落
- 12+：大成功，额外收益

有优势掷3D6取高两枚，有劣势取低两枚，两者同时存在就抵消。

# 五项能力

掷骰加上对应能力的调整值，范围-3到+3：

活力（力量、敏捷、体力）

沉着（冷静、专注、克制恐惧）

理性（研究、勘查、思维）

仪态（吸引、威慑、引发遐想）

敏感（超自然、神秘、灵异）

# 七种基本动作

| 动作     | 何时触发              |
| ------ | ----------------- |
| 日间动作   | 白天做危险的事           |
| 夜间动作   | 夜晚做危险的事，且事情比你想的更糟 |
| 插手动作   | 从某人或某地获取信息        |
| 休闲动作   | 休闲活动，与同伴分享亲密时刻    |
| 金冠之谜动作 | 讨论虚构小说系列，即兴编故事找灵感 |
| 神秘动作   | 使用超自然力量           |
| 推理     | 公开讨论，提出谜案解决方案     |

# 皇冠

掷骰结果出来后，你可以选择戴上皇冠，把成功等级提升一档。

代价二选一：

女王之冠：叙述一段情感回忆或现时场景，关于你的生命历程。

虚空之冠：按顺序标记腐化条目，逐步堕落，标记最后一项角色退休。

# 状况

状况让你掷骰处于劣势，最多3个。获得第4个状况时也必须戴皇冠。

# 温馨小天地

你家中那些有趣独特的物品。标记使用可获得掷骰优势，但标记后不再提供奖励，物品仍在，只是融入剧情。

# 例子

你深夜潜入嫌疑人的书房翻找文件。守秘人说事情比你想的更糟，楼上传来了脚步声。你掷夜间动作，沉着+1，2D6掷出4和2，总共7。常规成功：你找到了一份关键文件，但脚步声已经到了楼梯口，你被堵在房间里了。

你选择戴上女王之冠，提升到困难成功。你描述了一段闪回：你已故丈夫曾在某个深夜也是这样偷偷为你准备惊喜。然后你重新描述结果，你拿到了文件，还在脚步声到达前从窗户脱身。
