# 虚空之冠

当你戴上这顶王冠时，依次标记下列选项。

[ ] **花园中的阴影**

从现在起，在以你为中心的温馨场景或涉及你的休闲动作场景中，你还需要描述出黑暗实体如何在场景中微妙地出现。

[ ] **马车**

你的理性调整值减少1，你的敏感调整值增加1。

[ ] **苍白面具**

从现在起，在与任何其他角色进行任何亲密对话时，你必须不经意间提到死亡、垂死、来世或万物终结——无论当前的话题是什么。

[ ] **石榴籽**

你获得状况：沉迷于虚空。它永远不会被清除。

[ ] **虚空**

以一种沉迷于虚空的方式让你的角色退休。
